# resource-zh-site

## 项目简介

本项目是一个通用型 HTML 页面归档仓库，用于集中存放和发布多个独立的网页资源。仓库内容不绑定任何特定域名或网站，仅作为静态页面的存储与分发入口。

## 目录说明

```
resource-zh-site/
├── index.html          # 仓库首页 / 页面索引
├── pages/              # 归档的独立 HTML 页面
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
└── assets/             # 共用资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下存放各独立 HTML 页面，每个页面可包含自有的样式与脚本。
- `assets/` 目录存放页面间共享的静态资源，便于统一管理。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档文件，不依赖外部服务或特定域名。页面内容可能涉及信息展示、工具页面、参考文档等，具体用途请参考各页面内的说明。

页面文件命名清晰，便于通过 URL 直接访问。如需新增归档页面，请遵循以下规范：

- 页面文件置于 `pages/` 目录下。
- 文件名使用小写英文及连字符（如 `example-page.html`）。
- 页面内部建议包含简单的标题与内容说明。

## 使用方式

可直接通过 GitHub Pages 或其他静态托管服务部署本仓库，访问 `index.html` 即可查看页面索引。各独立页面可通过 `pages/` 目录下的文件名直接访问。
